## Naming convention

- 93 maps: `93mapMK{N}(PD).bin` where N is the revision number. MKF = Final.
- E30 maps: `E30mapMK{N}(PD).bin`
- E85 maps: `E85mapMK{N}(PD3M).bin`
- Flex maps: `MKMM({variant}).bin`
- Modified (93 folder): `Modifiedrev{N}.bin`
- `(PD)` = PID without I factor, `(PID)` = full PID tuning, `(PD3M)` = PD 3-map variant

## Context: two-machine setup

This repo is used on both Mac and Windows (TunerPro). The scripts handle both platforms:
- Mac: `refresh_shortcuts.sh` creates symlinks in the root folder
- Windows: `refresh_shortcuts.bat` creates `.lnk` shortcut files in the root folder

Neither symlinks nor `.lnk` files should be committed — they're local only, gitignored, and regenerated by the scripts.

## E85 note

Mac has E85 maps MK14–MK21 that Windows doesn't have yet. After you push from Mac, the Windows machine just needs to `git pull` and run `refresh_shortcuts.bat` to get up to date.
